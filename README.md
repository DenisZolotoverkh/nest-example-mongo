## Description

Nest mongo starter project

## Installation

```bash
# install dependencies
$ yarn install
```

```bash
# configure git hooks
$ yarn init-hooks
```

## Running the app

```bash
# development
$ yarn run start
```

```bash
# watch mode
$ yarn run start:dev
```

```bash
# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test
```

```bash
# test coverage
$ yarn run test:cov
```
